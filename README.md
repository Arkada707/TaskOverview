# Vite Vue Starter

This is a project template using [Vite](https://vitejs.dev/). It requires [Node.js](https://nodejs.org) v12+.

To start:

```sh
npm install
npm run dev

# if using yarn:
yarn
yarn dev
```

# End to end test

- Open cypress

```sh
npm run cypress:open
```

- Select E2E Testing
- Choose a browser
- Select Specs (e.g. tasks.cy.js)
